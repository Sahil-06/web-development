HTML & CSS Basics Summary

HTML Summary

HTML stands for Hyper Text Markup Language. It is the standard markup language for creating web pages. You don't use it for styling purposes.

  • Html element uses "elements" to describe (annotate) content
  • Html element typically having an opening tag, content
  • YOu can also have void element like images
  • There is a long list available elements but you will gain experience over time, no worries

Learn more about Html elements on MDN Html element reference.

CSS Summary

CSS (Cascading Style Sheet )is used for styling your page content.

  • Styles are assigned via property-value pairs
  • You can assign styles via the "style" attribute
  • To avoid code duplication, you typically use global styles (via the "style" element)
  • When working with CSS, concept like "inheritance", "specificity"and the "box model"should be understood

    • Learn about more css property on the MDN CSS property reference.
